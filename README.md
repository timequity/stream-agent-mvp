# stream-agent-mvp

Internal project name: `stream-agent-mvp`.

This repo supports the first Live Lab of the public format:

**"Справится ли AI? / Can AI Handle This?"**

The first experiment asks:

> Can AI help build an MVP assistant for streaming?

## MVP Scope

The MVP is a small, reversible container for testing an AI-assisted streaming workflow:

- OBS as the execution layer for live production.
- AI assistant for planning, context, and small support tasks.
- Stream summaries and post-stream notes.
- Speech-to-text experiments for capturing live input.
- Voice experiments for possible assistant responses.

The project starts as documentation and operating structure. Implementation comes only after the stream criteria are clear.

## Public Format

Each episode follows the same public mechanic:

task → criteria → AI attempt → human review → scorecard → verdict

Stream 001 is Episode 0 / Live Lab for checking whether AI tools can help create the MVP container without turning the stream into an unfocused dev session.

## Not Now

- No production build.
- No package setup.
- No API clients.
- No dashboard.
- No automation service.
- No secrets, tokens, stream keys, or private URLs.
- No full OBS integration yet.
- No voice system beyond planning experiments.
